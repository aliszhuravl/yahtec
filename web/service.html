Проектирование

Наша компания оказывает комплекс услуг по проектированию систем отопления и вентиляции, с применением газового промышленного оборудования YAHTECRUS, для различных объектов.

Мы можем осуществить:

  • Предпроектные исследования
  • Подготовку данных для получения технических условий на подключение к сетям газопровода
  • Разработку проектной документации
  • Согласование и получение положительного заключения экспертизы проектной документации
  • Разработку рабочей документации

Для проектных организаций, занимающихся проектированием систем с применением нашего оборудования, мы бесплатно предоставляем консультации и рекомендации:

  • По расчетам требуемых тепловых нагрузок
  • По подбору оборудования
  • По выбору технических решений
